## owl_architecture
# Запуск: 

Склонируйте репозиторий:
`git clone https://github.com/MistressHorse/owl_architecture.git`

Переместите локальный проект в папку scan.file

Установите зависимоти через команду: 
`pip install -r requirements.txt`

Запустите сканер командой: 
`python3 scan.py scan_file agressive # выберете тип агрессивности: medium, agressive`

Посмотрите результат сканирования в папке audit_json
